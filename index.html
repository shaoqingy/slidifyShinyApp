Project Overview

  1. This is a simple shiny application for the project of coursera course "Developing Data Products".

  2. The application aimts at predicting the diamond price according to its mass.

  3. The regression model was chosen for the prediction based on the training of the data "diamond" from the package "UsingR".

  4. You can find the application under the link: https://shaoqingy.shinyapps.io/myshinyapplication/

Project Operation

  1. On the left side, the diamond mass in carat could be chosen for prediction

  2. On the right side, the graphic displays the real prices and predicted prices for different masses and also the linear regression.

  3. The predicted price for the given mass would be shown in the graphic with black point.

  4. The predicted price would also be shown below with numbers.

  5. There is also a simple documentation for the application below.
